# 2.2 Event Table

| Event ID | Event Name | Trigger | Source | Activity | Response |
|----------|------------|---------|--------|----------|----------|
| EVT-001 | User Joins Room | User selects/enters room | Client App | Join Socket Room | Broadcast join confirmation |
| EVT-002 | Message Sent | User clicks send button | Client App | Encrypt & Encode Message | Send to server, save locally |
| EVT-003 | Message Received | Socket message event | Server | Decode & Decrypt Message | Display in chat, play sound |
| EVT-004 | Typing Started | User types in input | Client App | Emit typing start | Show typing indicator to others |
| EVT-005 | Typing Stopped | User stops typing (timeout) | Client App | Emit typing stop | Hide typing indicator |
| EVT-006 | Settings Changed | User updates username/key/room | Client App | Save to LocalStorage | Update UI and reconnect if needed |
| EVT-007 | Image Selected | User selects image file | Client App | Validate file size | Enable send button |
| EVT-008 | Secret Mode Toggle | User triple-clicks header | Client App | Switch between cover/chat mode | Change UI view |
| EVT-009 | Message History Loaded | Room changes | Client App | Query IndexedDB | Display saved messages |
| EVT-010 | Connection Lost | Network disconnect | Browser | Attempt reconnection | Show offline status |

## Notes
- All events are handled asynchronously where appropriate
- Local storage events are synchronous
- Socket events are real-time and bidirectional
