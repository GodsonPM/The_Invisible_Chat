# 2.1 Fact Finding Techniques

## Introduction
Fact finding techniques were employed to gather requirements and understand the needs for the Invisible-Chat application. The following techniques were used:

## 1. Record Review
- Reviewed existing chat applications and their security features
- Analyzed steganography research papers and implementations
- Studied WebSocket/Socket.IO documentation for real-time communication
- Examined encryption standards (AES) and best practices

## 2. Observation
- Observed user behavior on social media platforms like Instagram/Snapchat
- Noted the need for private communication channels
- Identified the potential for covert messaging in public feeds

## 3. Interviews
- Conducted self-interviews to define functional requirements:
  - Need for real-time messaging
  - Requirement for message encryption
  - Necessity of steganography for hiding messages in images
  - Local storage for message history

## 4. Questionnaires
- Created internal questionnaires to prioritize features:
  - Room-based chat (multiple channels)
  - User authentication via secret keys
  - Image-based message hiding
  - Typing indicators for better UX

## Conclusion
These techniques helped establish the core requirements: a secure, real-time chat system disguised as a social media feed using advanced steganography and encryption methods.
